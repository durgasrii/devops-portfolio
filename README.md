# DevOps Portfolio Deployment Project

## Project Overview

This project demonstrates a complete DevOps CI/CD workflow using GitHub, Jenkins, GitHub Webhooks and Netlify.

A professional portfolio website was developed using HTML, CSS and JavaScript and deployed on a public hosting platform. The project automates code integration, build verification and deployment through Jenkins and Netlify.

---

## Jenkins Pipeline Stages

### Stage 1 – Checkout

Jenkins fetches the latest source code from GitHub.

### Stage 2 – Verification

Pipeline validates the project files.

### Stage 3 – Build Success

Jenkins confirms successful execution.

### Stage 4 – Deployment

Jenkins triggers a Netlify Build Hook which automatically deploys the latest version of the website.

## Demonstration Scenario

1. Modify website content.
2. Save changes.
3. Push code to GitHub.
4. GitHub Webhook triggers Jenkins automatically.
5. Jenkins executes the pipeline.
6. Jenkins triggers Netlify deployment.
7. Updated website becomes available through the public URL.
